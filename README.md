# 🌐 Social Media Platform

A modern and interactive social media web application where users can connect, share posts, like, comment, and chat with others in real-time.

---

## ✨ Features

✅ User Registration & Login  
✅ Create, Edit & Delete Posts  
✅ Like & Comment System  
✅ Follow / Unfollow Users  
✅ Real-Time Chat  
✅ User Profile Page  
✅ News Feed  
✅ Notifications  
✅ Responsive Design  
✅ Dark Mode  

---

## 🛠️ Tech Stack

**Frontend:** HTML, CSS, JavaScript / React.js  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Authentication:** JWT / Firebase Auth
